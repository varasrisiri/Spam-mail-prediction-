# 📧 Spam Mail Prediction using Machine Learning

## 📌 Project Overview

This project classifies SMS messages as **Spam** or **Ham (Not Spam)** using Machine Learning. The model uses **TF-IDF Vectorization** for text feature extraction and **Logistic Regression** for classification.

---

## 🎯 Problem Statement

The objective of this project is to automatically identify whether an SMS message is spam or legitimate based on its text content.

---

## 📂 Dataset

The dataset used in this project is included in this repository as `mail_data.csv`.

Original Source:
https://www.kaggle.com/datasets/uciml/sms-spam-collection-dataset
